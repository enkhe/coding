# CSS Frameworks

> Decision: utility-first (Tailwind), component library (Bootstrap), or preprocessor (SCSS/Less).

## Comparison

| | Tailwind | Bootstrap | SCSS |
|---|---|---|---|
| Style | Utility classes in markup | Component classes (e.g., `.btn-primary`) | Preprocessor: nesting, mixins, `@use` |
| Bundle size | Tiny after purge | Moderate (kit) | Whatever you write |
| Customization | Theme via `tailwind.config` | Sass variables override | Anything |
| Best for | Modern apps, design systems | Quick admin/back-office | Existing Sass codebases |
| Combines with | Components + headless UI | jQuery (legacy) | Any framework |
