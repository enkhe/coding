# LeetCode

> Pattern practice. **NeetCode 150 / Blind 75** is the focused list to start.

## Topic strategy

| Pattern | Example problems |
|---|---|
| Two pointers | Two Sum II, 3Sum, Trapping Rain Water |
| Sliding window | Longest Substring Without Repeating, Min Window Substring |
| Hash map | Two Sum, Group Anagrams, Subarray Sum Equals K |
| Stack | Valid Parens, Daily Temperatures, Largest Rectangle in Histogram |
| Heap | Kth Largest, Merge K Sorted Lists, Top K Frequent |
| Binary search | Search in Rotated, Median of Two Sorted Arrays |
| BFS / DFS | Number of Islands, Word Ladder, Course Schedule |
| Backtracking | N-Queens, Word Break II, Sudoku Solver |
| DP | Climbing Stairs → House Robber → Coin Change → LCS → Edit Distance |
| Graph | Dijkstra, Topological Sort, Union-Find |
| Greedy | Jump Game, Gas Station |
| Bit manipulation | Single Number, Subsets via bits |

## Method

1. **Restate** the problem in your own words
2. **Brute force** first — confirm understanding
3. **Optimize** — apply pattern
4. **Edge cases** — empty, single, duplicates, negatives, max int
5. **Test mentally** before submitting
