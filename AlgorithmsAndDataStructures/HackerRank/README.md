# HackerRank

> Domain-by-domain progression. Good for SQL, Bash, and language-specific challenges.

## Useful tracks

- **Problem Solving** — algorithms + data structures
- **SQL** — joins, subqueries, window functions, advanced
- **Bash** — text processing, awk/sed
- **Regex** — pattern matching at scale
- **C#** / **Python** / **Java** — language-specific syntax practice

## Progression

1. Easy → Medium → Hard within a topic, not across topics
2. Time-box: 30 min for easy, 60 min for medium, 90 min for hard
3. After timeout, study the editorial; then re-implement from scratch the next day
