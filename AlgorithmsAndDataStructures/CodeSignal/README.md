# CodeSignal

> Time-boxed assessments + problems. Used by some companies for screening (General Coding Assessment).

## Test types

- **Arcade** — gamified practice
- **General Coding Assessment (GCA)** — 70 minutes, 4 problems graded by correctness
- **Industry Coding Framework (ICF)** — 4 hours, 1 large problem

## Strategy for GCA

1. Read all 4 problems first; rank by difficulty
2. Easy first → confidence + score floor
3. Skip if stuck > 5 min on the harder ones; come back at the end
4. Test edge cases before submitting (empty, single, max bound)
