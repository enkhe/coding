# Sandbox

> Throwaway-friendly experiments. Optimize for speed of learning, not polish.

## Conventions

- Folder name: `YYYY-MM-DD-<topic>/`
- Each folder contains:
  - `NOTES.md` — what I tried, what worked, what didn't
  - `RESULT.md` — one-paragraph conclusion
- Delete after 90 days unless promoted to `Demos/` or `Portfolio/`

## What sandbox is for

- "Does API X work the way I think?"
- "Can I get this 3rd-party library working?"
- "What's the perf if I do it this way?"
- "Reproduce a bug"

## What sandbox is NOT for

- Real work (use a feature branch in the actual repo)
- Production code (promote to `Portfolio/`)
- Long-lived experiments (graduate or delete)
